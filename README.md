Content:
1) main.py . Code that loads the dataset, computes epistemic hedge type distributions per each of the register and is able to produce figure 1,
which is a grouped bar chart(The_Hedge_Type_Dist.png) which compares Academic vs. News hedge type proportions.
2) Dataset.xlsx .  Its the dataset of 84 sentences which ahve been manually extracted and filtered from the Corpus of Contemporary American English
  COCA, Academic and News sub-corpora, filtered to 2019.
3) The_Hedge_Type_Dist.png , Figure 1 is a grouped bar chart output created from the code, which shows the distribution.
